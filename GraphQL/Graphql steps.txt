4. go run github.com/99designs/gqlgen init

5. go mod tidy

6. Do changes in the schema.graphqls

7. go run github.com/99designs/gqlgen generate

8. run this commands to get dependencies

-go get github.com/99designs/gqlgen/internal/imports@v0.17.30
